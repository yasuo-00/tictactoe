# Tic Tac Toe
Simple tic tac toe game made in Python
Minimax with Alpha-Beta Pruning implementation for computer AI

# Dependencies<br/>
-Numpy<br/>
-Pygame<br/>

To install dependencies with pip type
```
$ pip3 install numpy
$ pip3 install pygame
```
To execute from root folder
```
$ python3 src/main.py
```
